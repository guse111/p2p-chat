# P2P Chat

Децентрализованный P2P мессенджер на Python с использованием asyncio и ООП, работающий в локальной сети.

## Особенности

- **Полностью децентрализованный** — каждый пользователь является и сервером, и клиентом
- **Асинхронная архитектура** — параллельная отправка и получение сообщений
- **История сообщений** — связанный список с возможностью просмотра последних N сообщений
- **Простой CLI-интерфейс** — работа в терминале

## Технологии

- Python 3.14
- asyncio (async/await)
- TCP сокеты
- ООП (dataclass, ABC, композиция)
